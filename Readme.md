# Data Compression Collection
A repository of compression algorithms for different types of data (Including my own custom ones!)


## Why is this here?
I wanted to learn more about data compression! I thought a good way to understand and learn about data compression is to remake them myself in python and even try and take concepts and make my own modifcations of existing ones to fix flaws or maybe discover a new better way to compress data. This repo may also jsut contain some joke ones or odd omnes just for fun or to try random ideas I had.


## Can I contribute?
Maybe in the future i'll open up pull requests but for now this is just a showcase for my own work.


## How can I try them out for myself?
All compression systems will be designed as individual python modules that should be plug-and-play with any python project you may create and also feel free to credit me in any projects that you use my custom algorithms in! I'd love to see what you make with them!


## Was this written with ai?
All algorithms will not be written by ai as the point of this repo is for me to learn. AI may be used sometimes for the process of finding and researching compression algorithms along with wikipedia for me to find ones to recreate myself.
